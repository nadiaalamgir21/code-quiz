
# Code Quiz 

  

This is a Q&A project where each questions have multiple choices and when the user picks the wrong answer it will deduct 10 seconds from the overall time provided to finish the quiz. In the end the score will be the remaining time left after taking the quiz. If the user gets the answer correct, it will proceed to the next question. Ones the quiz is completed, the application will ask for initials to store the scores and the highest score will always be at the top and highlighted. Just for some fun I added dramatic clicker sound affects for correct and incorrect answers.
